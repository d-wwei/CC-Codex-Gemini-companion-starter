# Recommended MCP and Skills

This file is the Tier 3 catalog. All entries here are recommended and self-managed unless explicitly promoted into Tier 1 later.

## Infrastructure

- Feishu official MCP
  - Repository: <https://github.com/larksuite/lark-openapi-mcp/tree/main>
  - Cost: free
  - Good for document automation, conversations, and calendar workflows

- GitHub MCP
  - Cost: typically free
  - Good for repository, issue, and pull request workflows
  - Usually needs `GITHUB_TOKEN`

## Agent Enhancement

- Vercel agent-browser
  - Repository: <https://github.com/vercel-labs/agent-browser>
  - Cost: free
  - Strong browser-use option for reducing token-heavy browsing loops

- Sequential Thinking
  - Repository: <https://github.com/modelcontextprotocol/servers>
  - Cost: free
  - Useful when you want stronger stepwise reasoning support

- OpenMemory (Mem0)
  - Repository: <https://github.com/mem0ai/mem0-mcp>
  - Cost: local mode is free; cloud sync has a free tier
  - Useful for cross-session memory outside the native project memory layer

- Tavily MCP
  - Repository: <https://github.com/tavily-ai/tavily-mcp>
  - Cost: 1,000 free searches per month
  - Useful for structured web search and retrieval

- Exa MCP
  - Repository: <https://github.com/exa-labs/exa-mcp-server>
  - Cost: 1,000 free search requests per month
  - Useful for search augmentation and discovery

- Firecrawl MCP
  - Repository: <https://github.com/firecrawl/firecrawl-mcp-server>
  - Cost: 500 free crawls per month
  - Useful for dynamic pages, PDFs, and readable extraction

- Jina Reader
  - Repository: <https://github.com/jina-ai/MCP>
  - Cost: 10,000 free calls per month
  - Useful for turning noisy web pages into clean Markdown

## Finance

- Anthropic financial-services-plugins
  - Repository: <https://github.com/anthropics/financial-services-plugins>
  - Cost: free
  - Useful for finance-oriented workflows and integrations
